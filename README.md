# 📱 Портфолио Разработчика

Профессиональный шаблон портфолио для веб-разработчика с современным дизайном и адаптивностью.

## 🎨 Особенности

- ✨ Современный, минималистичный дизайн
- 📱 Полная адаптивность (мобильные, планшеты, десктоп)
- 🎯 Быстрая загрузка
- 🔍 SEO оптимизированный
- 🎭 Гладкие анимации и переходы
- 📧 Интегрированная форма контактов

## 📁 Структура файлов

```
portfolio/
├── index.html      # Главная страница
├── styles.css      # Стили
├── script.js       # Скрипты
└── README.md       # Этот файл
```

## 🚀 Быстрый старт

1. Откройте `index.html` в браузере - портфолио готово к работе!
2. Отредактируйте содержимое в соответствии с вашей информацией

## ✏️ Как отредактировать портфолио

### 1. Основная информация
В файле `index.html` найдите и измените:

```html
<h1>Привет! Я Разработчик</h1>
<p class="subtitle">Создаю современные и функциональные веб-приложения</p>
```

### 2. Обо мне
Обновите секцию "Обо мне":

```html
<p>Я опытный веб-разработчик...</p>
```

### 3. Ваши навыки
Отредактируйте карточки навыков в секции "Мои навыки". Измените иконки:
- 🔧 Используйте иконки из [Font Awesome](https://fontawesome.com)
- Примеры: `fa-code`, `fa-server`, `fa-database`, `fa-tools`

### 4. Ваши проекты
В секции "Мои проекты" отредактируйте:
- Название проекта
- Описание
- Технологии (теги)
- Ссылки на проекты

### 5. Контактная информация
Обновите в секции "Давайте свяжемся":
- Email: замените `your.email@example.com`
- Телефон: замените `+7 (999) 999-99-99`
- Город: укажите вашу локацию

### 6. Социальные сети
Измените ссылки в социальных сетях на свои профили:

```html
<a href="https://github.com/yourprofile" title="GitHub"><i class="fab fa-github"></i></a>
```

## 🎨 Цветовая схема

Основные цвета можно изменить в `styles.css`:

```css
:root {
    --primary-color: #6366f1;      /* Основной цвет */
    --secondary-color: #ec4899;    /* Вторичный цвет */
    --dark-bg: #0f172a;            /* Темный фон */
    --light-bg: #f8fafc;           /* Светлый фон */
    --text-dark: #1e293b;          /* Темный текст */
    --text-light: #64748b;         /* Светлый текст */
}
```

## 🌐 Развертывание

Вы можете легко развернуть портфолио на:

### GitHub Pages
1. Загрузите файлы в репозиторий
2. Перейдите в Settings → Pages
3. Выберите ветку `main` как источник
4. Ваше портфолио будет доступно по адресу: `https://yourusername.github.io/portfolio`

### Netlify
1. Перетащите папку на [netlify.com](https://netlify.com)
2. Готово! Получите уникальный URL

### Vercel
1. Импортируйте репозиторий
2. Автоматически развернется
3. Получите URL для доступа
